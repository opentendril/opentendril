# Global Naming Conventions

This project strictly adheres to specific naming conventions to ensure the architecture matches the biological OpenTendril metaphor.

1. **File Names:** All files MUST be named using `kebab-case` (e.g., `my-file-name.py`). You are strictly forbidden from using underscores (`_`) or camelCase in filenames.
2. **Biological Parity:** Always prioritize using botanical terminology (stem, tendril, root, sprout, senesce, elongation) for system architecture components.
