# Code of Conduct

## Our pledge

We want OpenTendril to be a welcoming, respectful, and collaborative project.
Everyone who participates in the project is expected to help create a
harassment-free experience for all contributors, regardless of background,
identity, experience, or role.

## Expected behavior

- Be respectful, constructive, and considerate.
- Assume good intent while addressing the impact of harmful behavior.
- Give and receive technical feedback professionally.
- Respect different viewpoints, goals, and levels of experience.
- Keep project discussions focused on the work and accessible to others.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Sexualized language or attention that is unwelcome.
- Trolling, deliberate disruption, or sustained bad-faith participation.
- Publishing private information without permission.
- Threats, intimidation, or other conduct that would be inappropriate in a
  professional community.

## Enforcement

Please report problems privately to the repository maintainers using GitHub's
private vulnerability reporting channel when the report concerns a security
issue. For other conduct concerns, contact the maintainers through a private
GitHub message or repository support channel. Reports will be handled
confidentially where possible.

Maintainers may remove comments, restrict participation, or take other action
they consider necessary to protect the community. They will aim to explain
moderation decisions when doing so does not compromise privacy or safety.

This Code of Conduct applies in project spaces and whenever someone is
representing the project in public.
